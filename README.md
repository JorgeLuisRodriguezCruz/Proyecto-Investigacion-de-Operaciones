# Algoritmos de Programación Dinámica

Aplicación interactiva para visualizar y ejecutar algoritmos de programación dinámica.

## Requisitos

- Node.js v18 o superior

## Instalación

```bash
npm install
```

## Ejecución

```bash
npm run dev
```

## Módulos

1. Reemplazo de Equipos
2. Árboles Binarios de Búsqueda Óptimos
3. Series Deportivas
4. Multiplicación de Matrices
